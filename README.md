# Minimalistic Express.js docker setup
Alpine based docker setup with Nginx, Node.js and Express.js.

## Setup includes:
* node:lts
* nginx:1.20.2-alpine
* expressjs: ^4.18.1

## Installation:
1. Copy files from this repo
2. In project directory run:
```
docker-compose up -d
```

## App url:
http://localhost:80/
